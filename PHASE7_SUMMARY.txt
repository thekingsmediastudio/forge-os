🚀 TO PRODUCE A RELEASE BUILD:

1. Generate a release keystore:
     keytool -genkey -v -keystore forge-release.jks \
       -keyalg RSA -keysize 2048 -validity 10000 -alias forge

2. Create app/keystore.properties (gitignored) with storeFile, storePassword,
   keyAlias, keyPassword.

3. Build:
     ./gradlew clean :app:bundleRelease
     # → app/build/outputs/bundle/release/app-release.aab

4. Verify ProGuard didn't strip @Serializable models:
     ./gradlew :app:assembleRelease && adb install -r app-release.apk

5. Upload AAB to Play Console internal track; promote when smoke-test passes.

See RELEASE.md for the full checklist.
